# 数学口算题生成器 — 开发文档

## 项目概述

一个纯前端网页应用，用于生成小学数学口算练习题PDF。用户选择题型、设置页数和每页题数，一键生成A4排版的PDF文件直接下载，可打印或保存。

- **线上地址**：https://xiaosong746-png.github.io/math-worksheet/
- **本地文件**：`math-worksheet/index.html`
- **技术栈**：HTML + CSS + JavaScript（单文件）
- **外部依赖**：jsPDF + html2canvas（CDN引入，用于PDF生成）
- **托管平台**：GitHub Pages（免费）

## 题库管理机制

### 预生成全量题库
- 页面加载时，枚举每种题型的所有合法算式组合
- 存入内存数组，Fisher-Yates洗牌打乱
- 用过的题目通过 `key` 值（如 `3+5`）追踪

### 选题逻辑
```
用户选择N种题型 + 总题数T
  → 每种题型分配 ≈ T/N 道
  → 各自从独立题库中抽取（不重复）
  → 全部抽完后混合洗牌
  → 按每页题数分页
```

### 去重与循环机制
- **页内去重**：每次生成从题库顺序抽取，同页不会重复
- **跨次去重**：已用题目存入 `localStorage`，下次生成时跳过
- **循环重置**：某题型题库不够时，自动重置并重新洗牌，继续抽取
- **多次循环**：如需500道但题库仅63道，则循环抽取63→重置→再抽63…直到凑够
- **静默运行**：全程无提示，用户无感知

### 存储位置
- **题库**：浏览器内存（每次打开页面重新生成）
- **已用记录**：浏览器 `localStorage`（key: `math_worksheet_used`）
- 清除浏览器缓存 = 已用记录归零

## PDF生成机制

### 方式
使用 **html2canvas + jsPDF** 直接生成PDF文件并触发浏览器下载。

### 流程
```
1. 生成HTML内容 → 放入隐藏的print-area
2. html2canvas渲染每个页面为高清图片（2倍缩放）
3. jsPDF创建A4文档，逐页添加图片
4. doc.save('数学口算题_2026-06-14.pdf') 触发下载
```

### 排版规则
- **列数**：固定4列
- **字号**：根据每页行数自动适配（10pt~18pt）
- **行高**：Flex布局自动撑满A4页面
- **字体**：宋体（SimSun），标题黑体（SimHei）
- **页边距**：上下8mm，左右12mm
- **每页上限**：100道

### 字号与行数对应

| 每页题数 | 行数 | 字号 |
|----------|------|------|
| 100 | 25行 | 10pt |
| 80 | 20行 | 11pt |
| 64 | 16行 | 12pt |
| 52 | 13行 | 14pt |
| 40 | 10行 | 16pt |
| ≤28 | ≤7行 | 18pt |

## 新增题型指南

如需新增题型，在 `index.html` 中修改三处：

### 1. 题型定义（TYPE_DEFS对象）
```javascript
newType: { name: '新题型名称', gen: 'genNewType', maxResult: 100 }
```

### 2. 生成逻辑（generateAllQuestions函数的switch语句中）
```javascript
case 'genNewType':
    for (let a = 0; a <= def.maxResult; a++) {
        for (let b = 0; a + b <= def.maxResult; b++) {
            questions.push({
                display: a + '运算符' + b + '=',
                key: a + '运算符' + b
            });
        }
    }
    break;
```

### 3. 界面复选框（type-grid区域）
```html
<label class="type-item" data-key="newType">
    <input type="checkbox">
    <span class="check-box"></span>
    新题型名称
</label>
```

## 常见问题

**Q: 为什么有些题型题库很小？**
A: 数学限制。如"20以内进位加法"只有约65种不重复组合，这是该规则下的全部可能。系统会自动循环抽取确保多页出题。

**Q: 题库存在哪里？数据安全吗？**
A: 全部在浏览器本地（内存 + localStorage），不上传任何数据到服务器。

**Q: 最多多少人同时使用？**
A: 无限制。每个人浏览器独立运行，GitHub Pages免费带宽足够。

**Q: 手机端文件名为什么也正确了？**
A: 使用jsPDF的`doc.save()`方法直接控制文件名，不再依赖浏览器打印功能。

**Q: 生成的PDF为什么是图片格式？**
A: html2canvas将HTML渲染为高清图片再嵌入PDF，确保中文完美显示、排版精确。
